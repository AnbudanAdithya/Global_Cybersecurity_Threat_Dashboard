# Global Cybersecurity Threat Dashboard

**Roll Number:** 24030302038
**Program:** Master of Business Administration (MBA)

---

## Project Overview

This project features an interactive Power BI dashboard designed to analyse and visualise global cybersecurity threats. By transforming raw threat data into actionable visual insights, this dashboard provides a comprehensive overview of attack vectors, geographic vulnerabilities, and threat severity, aiding in strategic risk management and data-driven decision-making.

---

## Repository Structure

* `24030302038_Dashboard.pbix`: The core Power BI dashboard file containing all interactive visualisations and data models.
* `24030302038_Report.pdf`: A detailed analytical report documenting the methodology, key findings, and strategic interpretations of the data.
* `Global_Cybersecurity_Threats.csv`: The primary dataset utilised for this analysis.

---

## How to Use

1.  **View the Findings:** Open the `24030302038_Report.pdf` for a complete breakdown of the analysis and conclusions.
2.  **Explore the Dashboard:** * Download `24030302038_Dashboard.pbix`.
    * Open the file using **Microsoft Power BI Desktop**.
    * Interact with the filters and visuals to explore the global threat landscape.

---

## Data Source

The analysis is based on the `Global_Cybersecurity_Threats.csv` dataset, which contains records of reported cyber incidents, including threat types, origins, and impact metrics. This dataset was sourced from [Kaggle](https://www.kaggle.com/datasets/atharvasoundankar/global-cybersecurity-threats-2015-2024)
